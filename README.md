## Usage

### 1. Download project

### 2. Change to Practice #1 folder directory

```js
cd cd nextjs-training
cd crypto-currency-news
```
### 3. Install packages
```js
npm install
```
### 4. Scripts

*You can use scripts below to work with this project*
```js
// run project on local server
npm run dev

// build project to product
npm run build

// run test
npm run test
```
